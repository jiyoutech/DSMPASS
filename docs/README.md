# 文档索引

当前开源主线是 Go 版本。

## 推荐阅读

- `go-version.md`：Go 版本结构和本地运行方式
- `spk-feishu-setup.md`：从安装 SPK 到配置飞书登录和通讯录同步的完整步骤
- `go-dsm-binary-deployment.md`：DSM 二进制部署方式
- `dsm-spk-package.md`：DSM SPK 安装包、升级、卸载和端口配置
- `provider-development.md`：新增身份源 provider 的开发边界
- `publication-guidelines.md`：哪些内容适合公开，哪些不能放到开源仓库
- `testing.md`：单元测试、前端构建和文档检查说明
- `release.md`：发布前检查清单

## 不公开的内容

内部产品设计稿、真实部署记录、敏感日志、密钥、数据库、客户/企业信息、详细攻击面分析和未脱敏排障材料不放入开源仓库。
